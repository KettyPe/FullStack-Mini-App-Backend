# FullStack Mini App — Backend
 
Учебный fullstack-проект. Серверная часть приложения на стеке MERN.
 
---
 
**Репозиторий фронтенда:** https://github.com/KettyPe/FullStack-Mini-App-Frontend
 
---
 
## О проекте
 
Первый опыт разработки собственного REST API.  
Бэкенд обслуживает клиентское приложение: авторизация, работа с базой данных, загрузка файлов.
 
### Что реализовано
 
- REST API на Express.js
- Регистрация и авторизация пользователей
- JWT-аутентификация и защита маршрутов
- Работа с базой данных MongoDB через Mongoose
- Загрузка файлов на сервер
- Деплой на Vercel
 
---
 
## Стек
 
- Node.js, Express.js
- MongoDB, Mongoose
- JWT
- JavaScript
 
---
